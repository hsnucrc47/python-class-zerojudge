# 經典河內塔題解
## 程式碼
```py
def hanoi(n, source, auxiliary, target):
    if n == 1:
        print(f"Move disc 1 from {source} to {target}")
    else:
        hanoi(n - 1, source, target, auxiliary)
        print(f"Move disc {n} from {source} to {target}")
        hanoi(n - 1, auxiliary, source, target)
 
n = int(input())
hanoi(n, 'A', 'B', 'C')
```

## 解說
```python
    if n == 1:
```

1. 設定終止條件，當遞迴到最小的 n 為 1 時，直接將 `disc 1` 從 `source` 移動到 `target` 。

```python
    else:

        print(f"Move disc {n} from {source} to {target}")
        hanoi(n - 1, auxiliary, source, target)
```
2. 觀察規律，最大的圓盤（ `n` ） 一定只會動一次，且移動從 `A` 移動到 `C` 。再比較小一號圓盤移動方式，將 `source` 、 `target` 、 `auxiliary` 的順序重新排列。
